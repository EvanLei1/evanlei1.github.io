Compatibility

This website is created using Github Pages. It was used to make the website more modern and open-source thanks to HTML5 and CSS3.

However, it was intended for use with newer browsers, yet those older browser versions are not compatible with this website.

Operating System Support

Windows

Browsers using the Windows 9x interface do not have support at all. Windows 3.x, supports up to Internet Explorer 5, Windows 95 as 5.5, and Windows 98 and ME as 6. Very few sites support Internet Explorer 6 as of 2024. Windows NT 4.0 is based on Windows 95 that supports up to IE6 while Windows 2000 is based on Windows 98/ME and also supports up to IE6. However, Windows XP supports IE8, Vista supports IE9, and Windows 7 and up supports IE11. Other browsers stopped updating them as soon as support for older Windows versions ended.
